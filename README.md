# INTEMASS AI: Automated Essay Grading System

## Overview

INTEMASS AI is a prototype for a next-generation automated essay grading system. It leverages modern AI to provide students and teachers with fast, consistent, and insightful feedback on written submissions.

This guide will walk you through the main features of the application.

## How to Use

This application has two primary user workflows: one for teachers creating questions and one for students submitting answers. You can navigate between different features using the sidebar on the left.

---

### 1. Dashboard

The **Dashboard** is the landing page of the application. It provides a high-level overview of system activity, including:
- Average scores and submission statistics.
- A chart showing the distribution of scores.
- A list of recent student submissions.

---

### 2. Essay Scoring

This feature allows you to get an instant AI-powered score for any essay, based on a marking scheme you provide.

**How to use it:**
1. Navigate to the **Essay Scoring** page from the sidebar.
2. Paste the student's essay into the "Essay Text" area.
3. Provide the marking criteria or rubric in the "Mark Scheme" area.
4. Click **Get AI Score**.
5. The system will return a score, along with detailed feedback, strengths, and weaknesses.

---

### 3. AI Feedback Generation

Generate personalized feedback for a student's essay without a formal scoring scheme. This is useful for formative assessments.

**How to use it:**
1. Go to the **AI Feedback** page.
2. Enter the student's grade level and the essay topic.
3. Paste the full essay text.
4. Click **Generate Feedback**.
5. The AI will provide a report identifying strengths, weaknesses, and actionable suggestions for improvement.

---

### 4. Diagram Analysis

Analyze diagrams, charts, or drawings within an essay. The AI can check for completeness and accuracy against a set of expected components.

**How to use it:**
1. Go to the **Diagram Analysis** page.
2. Upload an image of the diagram. A preview will be shown.
3. In the "Expected Components" text area, list the parts you expect to see in the diagram, separated by commas (e.g., *Nucleus, Mitochondria, Cell Wall*).
4. Click **Analyze Diagram**.
5. The AI will provide a completeness score and feedback on the diagram's accuracy.

---

### 5. OCR for Handwritten Submissions

Convert a handwritten or scanned essay into digital text using Optical Character Recognition (OCR).

**How to use it:**
1. Navigate to the **OCR Submission** page.
2. Click to upload an image of the handwritten essay.
3. The system will automatically process the image and display the extracted text in the adjacent text area.
4. You can then copy the extracted text to use in the Essay Scoring or AI Feedback tools.

---

### Teacher-Specific Workflows

The following pages are designed for teachers to manage the question bank.

#### Create a New Question
1. Go to **Create Question**.
2. This section will guide you through a multi-step process to define the question, the standard (ideal) answer, and the marking scheme. *(Note: This is a feature in development).*

### Student-Specific Workflows

The following pages are designed for students to submit their work.

#### Submit an Answer
1. Go to **Submit Answer**.
2. Here, you can browse available questions and select one to answer. *(Note: This is a feature in development).*

#### View Results
1. Go to **View Results**.
2. This page will display a history of your graded submissions, allowing you to review your scores and feedback. *(Note: This is a feature in development).*
